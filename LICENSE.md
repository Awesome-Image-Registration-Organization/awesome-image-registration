# **Open Source Data(OSD)协议**


本项目数据主要是“nCov疫情地图”团队（下简称为nCov团队）成员在疫情期间采集和收集处理的，版权归nCov团队所有。

为了能够让更多的人对数据进行分析研究，为疫情提供服务，故将数据采用OSD协议进行开源，开源规则如下：

1. 署名。本数据可用于学术研究，发表学术成果，但**必须标明数据来源**，在相关成果中引用本项目的GitHub地址；
2. 非商业授权。本数据主要通过互联网进行采集整理而来，主要为疫情提供服务和决策支持，任何企业或者组织**不能用于商业性质活动**；
3. 相同方式共享。使用本项目数据集，对数据进行了更新或者在基础上产生了新的数据集，同样需要无偿开源出来，并进行声明；
4. 免责。如果其他团队或企业使用我们数据进行不正当行为，nCov团队不承担任何法律责任。
